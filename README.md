# MyClient

这是一个名为 MyClient 的前端项目，使用 Vue 3 和 Vite 构建。

## 项目设置

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 项目结构

- `src/` - 源代码目录
  - `assets/` - 静态资源（CSS、图片等）
  - `components/` - Vue 组件
  - `App.vue` - 主应用组件
  - `main.js` - 应用入口文件
- `public/` - 不需要通过构建工具处理的静态资源
- `index.html` - HTML 入口文件
- `vite.config.js` - Vite 配置文件

## 特性

- 使用 Vue 3 的组合式 API
- Vite 提供快速的开发体验
- 简洁的项目结构
- 响应式设计
